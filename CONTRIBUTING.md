# How to contribute

Thanks for taking the time to contribute! Bug reports, feature requests, code,
documentation, and tests are all welcome.

- This repository is a community-maintained fork of the original
  [Sensmetry SysIDE Editor](https://gitlab.com/sensmetry/public/sysml-2ls).
- File issues and pull requests on this GitHub repository.
- Developer setup instructions are in [README.md](README.md#building-from-source).

## Reporting bugs

Open an issue on GitHub. Please search existing issues first to avoid
duplicates, and include enough context to reproduce: a minimal SysML
snippet, the expected vs. actual behavior, and your VS Code / Node.js
versions.

## Suggesting features

Open an issue on GitHub describing the use case and the proposed behavior.
