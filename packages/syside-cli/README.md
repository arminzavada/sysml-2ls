# _SysIDE_ CLI

Standalone command-line interface for the _SysIDE_ language server.
Useful for parsing, validating, and formatting SysML v2 / KerML files
without a running editor.

See the top-level [README](../../README.md) for project context and
[`packages/syside-languageserver`](../syside-languageserver) for the
underlying language services.
