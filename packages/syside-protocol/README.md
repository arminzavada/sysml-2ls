# _SysIDE_ LSP Extensions

Custom LSP request/notification types shared by the _SysIDE_ language server
and any clients (e.g. the VS Code extension and CLI). Anything beyond the
standard LSP — for example, `FindStdlibRequest` — is defined here so the
two sides agree on the wire shape.

See the top-level [README](../../README.md) for project context.
